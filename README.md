
# 🎮 Multiplayer Game JavaScript (Socket.IO + Node.js)
# PvP
- Этот проект создавался с целью попрактиковатся с работой в JavaScript и создании технологии Multiplayer



## 📖 Остновная идея
- Основная идея игры - это 1v1 против другого человека, где выйти победителем может только один


## ✨ Основные возможности
# PvP (сражение против другого игрока(пока только голыми руками))
# Востановление здоровья (вы можете восстановить своё здоровье при помощи аптечек разбросаных по карте)



## 🛠️ Используемые технологии

В проекте используются следующие технологии:

- **JavaScript (ES6+)** — основной язык разработки.
- **Node.js** — серверная часть приложения.
- **Socket.IO / WebSocket** — организация связи в реальном времени.
- **HTML5** — структура пользовательского интерфейса.
- **CSS3** — оформление и стилизация.
- **npm** — управление зависимостями проекта.
