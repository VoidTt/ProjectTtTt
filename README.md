
# 🎮 Multiplayer Game JavaScript (Socket.IO + Node.js)
# PvP
- Этот проект создавался с целью попрактиковатся с работой в JavaScript и создании технологии Multiplayer



## 📖 Остновная идея
- Основная идея игры - это 1v1 против другого человека, где выйти победителем может только один(пока что не реализованна)
- на данный момент реализован только вход через логин или регистрацию


## ✨ Основные возможности
взаимодействие друг с другом
- регистрация + вход
- индивидуальные ники
- вход на сервер только по id запустившего сервер(Radmin.vpn maybe)



## 🛠️ Используемые технологии

- **JavaScript (ES6+)** — основной язык разработки;
- **Node.js** — серверная часть приложения;
- **Socket.IO — организация связи в реальном времени(реализация мультиплеера);
- **npm.cmd** — управление зависимостями проекта;
- **Three.js** - 3D;

## 🛠️ Как запустить сервер
для работы сервера нужно скачать:
Node.js + Three.js

после чего в папке ProjectTtTt в терминале вводим:
 - npm.cmd run dev
Если не запускается установите зависимости:
 - npm init -y
 - npm i express socket.io
 - npm i -D nodemon
и попробуйте снова
